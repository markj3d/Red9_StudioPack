remove keyed data from the ctrl_main and re-calculate the rest of the rig to compensate. The situation would be that the Main Ctrl has been used to do the bulk of the characters motion, think walk on the spot then using the Main to do the transforms in world space. We want to remove those Main transformations, reset the Main ctrl and recalculate the rest of the rig accordingly.

  • step â the bake step used IF not smartbake
  • smartbake â if True we use the smartbake systems when extracting key data
