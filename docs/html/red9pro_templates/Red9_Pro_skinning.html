Pro_Pack : skinning¶

ProPack skinning management and file format for Maya

>>> # import statement for the module via the r9pro decompiler
>>> from Red9.pro_pack import r9pro
>>> r9skn = r9pro.r9import('r9skn')

Code examples:

Examples

>>> # Import the tools
>>> r9pro.r9import('r9skn')
>>> from r9skn import *

>>> # Duplicate a mesh and load skin data in new mesh
>>> # ================================================
>>> # Get skin meshes in the scene
>>> skin_meshes = get_skin_meshes(selection=False)
>>> # convert a mesh to Skn object
>>> r9skn = Skn(skin_meshes[-1])
>>> # collect skin data
>>> skin_data = r9skn.get_data()
>>> # duplicate the mesh
>>> new_mesh = cmds.listRelatives(cmds.duplicate(r9skn.transform), ad=True, s=True, type='mesh', ni=True, f=True)[0]
>>> # Create a new Skn object with duplicated mesh
>>> new_r9skn = Skn(new_mesh)
>>> # Load weights in new meh
>>> new_r9skn.load(skin_data)

Creating Skin Files:

>>> # Create .skn files save and load
>>> # ================================================
>>> all_skins = 'file/path/to/file/all_skins.skn'
>>> selected_skins = 'file/path/to/file/selected_skins.skn'
>>> specific_skin = 'file/path/to/file/specific_skin.skn'
>>> 
>>> # Create a Skn_file object
>>> skn_file = Skn_file()
>>> # Create skin file with the data of all skinned meshes in the scene
>>> skn_file.save_file(file_path=all_skins, selection=False, meshes=None)
>>> # Create skin file with the data of selected skinned mesh or meshes, also works forcomponent selection
>>> cmds.select(get_skin_meshes()[-1])
>>> skn_file.save_file(file_path=selected_skins, selection=True, meshes=None)
>>> # Create skin file with an specific mesh/meshes
>>> meshes = get_skin_meshes()[-3:]
>>> skn_file.save_file(file_path=specific_skin, selection=False, meshes=meshes)

Deleting Skin Clusters:

>>> # delete all skin clusters in the scene
>>> for mesh in get_skin_meshes():
>>>     skn = Skn(mesh)
>>>     skn.delete_skin_cluster()
>>>     skn.delete_dag_poses()

Loading Skin File Data:

>>> # Create a Skn_file object
>>> skn_file = Skn_file()
>>> # load all skins from file
>>> skn_file.file_path = all_skins
>>> # Load data of specific mesh stored in the file
>>> skn_file.load(data=skn_file.data[mesh], selection=False)
>>> # reload skin data from file
>>> skn_file.read_skin_file(all_skins)
>>> # load all skins from file
>>> skn_file.load(selection=False)
>>> # Load on selected mesh or component
>>> cmds.select(mesh)
>>> skn_file.load(selection=True)

class Weights(r9skn)¶

Bases: object

Weight class to interact with skinning.Skn objects contains wrappers to deal with Maya API and R9skin format

>>> r9skn = r9pro.r9import('r9skn')
>>> skn_object = r9skn.Skn('|Transform|Shape')
>>> weights = r9skn.Weights(skn_object)
>>>
>>> # Collect weight data from skin cluster
>>> api_weights = weights.get_weights_from_skincluster()
>>>
>>> # Reset weights to default, use .skincluster attribute from skn_object
>>> cmds.skinPercent(skn_object.skincluster, rtd=True)
>>>
>>> # Set original weights on skin cluster
>>> weights.set_weights_to_skincluster()
>>>
>>> # Get weight data as dictionary, as use in skn file format
>>> r9_weights = weights.r9_weights
